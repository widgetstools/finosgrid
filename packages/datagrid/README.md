# @widgetstools/finosgrid

Perspective viewer datagrid plugin with AG Grid–like chrome.

Peer dependencies: `@finos/perspective`, `@finos/perspective-viewer` (^3.8).

```bash
npm install @widgetstools/finosgrid @finos/perspective @finos/perspective-viewer
```

```js
import "@widgetstools/finosgrid";
```

Registers the `perspective-viewer-datagrid` custom element and viewer plugin.
